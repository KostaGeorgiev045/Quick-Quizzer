How to Use

Load a JSON file with your quiz questions. Questions and answers are shuffled each attempt.

JSON Format

{
  "title": "My Quiz",
  "questions": [
    {
      "question": "Your question?",
      "answers": ["A", "B", "C", "D"],
      "correct": 1
    },
    {
      "question": "Multi-select?",
      "answers": ["A", "B", "C"],
      "correct": [0, 2]
    }
  ]
}

correct: index (single) or array of indices (multi-select)
